# 书生·浦语大模型全链路开源开放体系

## 大语言模型为什么具有较高的关注度？

大语言模型是发展通用人工智能（AGI）的重要途径，大语言模型促使人工智能的模型研究由单个的专用模型向通用模型发展（即使用单个模型去解决多个常见的场景，例如：音频、图像等）
